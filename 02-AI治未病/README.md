# AI "治未病" - 长期陪伴

→ 完整内容见：[AI治未病-长期陪伴.md](./AI治未病-长期陪伴.md)

## 内容摘要

**核心动机**：LLM 成熟后，AI 首次具备提供"情绪价值"的能力；"治未病"最大的敌人是依从性，AI 以"润物细无声"方式干预生活细节。

**技术路径**：
- **长程记忆**：Chain of Agents（CoA）框架——分块协作、接力传递、协同推理，解决"Lost in the middle"问题
- **RAG-LC 混合模型**：RAG 提供微观证据（医疗知识图谱检索），Long Context 进行因果推导
- **数字生物标志物**：语音微震颤、打字动力学、步态耦合分析——"秒检制"取代"年检制"
- **主动干预**：基于动机访谈逻辑的 Agentic Intervention，依从性提升约 51%
- **数字孪生**：Health Digital Twin，ETH 2026 可视化长期健康预测实验

**代表产品**：ElliQ 3.0（老年孤独感）、Headspace Ebb（职场焦虑 24/7）、Apple Health+ 2026 Beta（终身健康足迹）
