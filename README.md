## 核心主题速览

**可穿戴健康硬件（北美前沿）**
- 无袖带血压：PyrAmes Boppli、Aktiia Hilo Band、UCSD CircuCare 超声贴片
- 无针血糖：Biolinq Shine（2025.9 FDA De Novo 批准）
- 消费端里程碑：Apple Watch 高血压通知 2025.9 FDA 批准
- 环境感知：Emerald 无线射频家庭监测（步态/呼吸/跌倒）
- FDA 趋势：2024–2025 是消费级健康硬件监管分水岭

**AI 长期陪伴技术**
- 长程记忆：Chain of Agents（CoA）框架解决上下文爆炸
- 数据整合：RAG-LC 混合模型（外部知识检索 + 长窗口因果推理）
- 主动干预：基于动机访谈（Motivational Interviewing），依从性提升约 51%
- 数字孪生：Health Digital Twin（ETH Zurich 2026 实验）

**市场与商业化**
- AI 医疗真正壁垒：FDA 合规资质、医院系统对接（Epic 生态）、垂直场景深度 knowhow
- 2B 优于 2C：能节省医生 2–3 小时/天的产品（如 Abridge）渗透率快速提升
- CGM 市场：消费级 Stelo（Dexcom）vs Lingo（Abbott），OTC 无需处方
- 创新方向：中医+AI、玄学融合，低门槛情绪价值产品
