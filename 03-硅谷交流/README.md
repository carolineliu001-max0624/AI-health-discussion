# 硅谷交流 Takeaway - AI Health

→ 完整内容见：[硅谷交流takeaway-AI_health.md](./硅谷交流takeaway-AI_health.md)

## 内容摘要

**CGM 血糖监测市场**：
- 消费级 OTC：Stelo（Dexcom，与 Oura 官方集成）vs Lingo（Abbott，iOS only）
- 对标医疗级：Dexcom G7 / FreeStyle Libre 3（支持胰岛素泵、实时临床决策）

**AI 医疗产品 vs AI Chatbot 的差异**：
- Chatbot 缺乏主动监控、缺乏个人 context、缺乏数据闭环
- 真正壁垒：FDA 合规资质 + 医院系统对接 + 垂直场景 knowhow

**美国 AI 医疗现状**：
- 医生文书负担重（2–3 小时/天），Abridge 自动生成 EHR 渗透率快速提升
- Epic 垄断 EHR 市场，数据封闭是 AI 产品获取 context 的主要障碍
- 犹他州允许 AI 开具处方（突破性但尚未推广）

**2C vs 2B**：2C 缺付费方、技术壁垒低；2B 需深度合规与专科适配，成功案例须显著提效

**创新方向**：
- 中医+AI：脉象监测手环、心理与家庭医生替代场景
- AI+玄学：与算命/风水/生肖八字结合，低门槛情绪价值产品
